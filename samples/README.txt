Sandesha Samples.

If you have downloaded the source distribution, Please build the source using maven first.

Following ant targets are availble to test Samples.

ping_sync       --> runs sample test for AsyncPing
ping_async      --> runs sample test for SyncPing
echo_sync_ack   --> runs sample test for EchoStirng
echo_async_ack  --> runs sample test for EchoStirng
clean           --> cleans up all generated files and
usage           --> provides help on using the build tool(default)

For more information on Apache Sandesha usage, please refer to the Sandesha User Guide.
